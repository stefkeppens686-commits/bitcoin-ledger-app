# Installation Guide

## Building for Ledger

This guide covers how to build and install the Bitcoin Ledger app on your Ledger hardware wallet.

### Requirements

- Ledger Nano S, Nano S Plus, or Nano X
- Ledger Live (latest version)
- Development environment setup

### Steps

1. **Build the application**
   ```bash
   npm run build
   ```

2. **Load onto device**
   Follow official Ledger documentation for sideloading applications.

3. **Verify installation**
   - Connect your Ledger device
   - Open Ledger Live
   - Navigate to the Bitcoin app
