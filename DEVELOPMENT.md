# Development Guide

## Prerequisites

- Node.js 16 or higher
- npm or yarn
- Ledger development tools

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/stefkeppens686-commits/bitcoin-ledger-app.git
   cd bitcoin-ledger-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the project:
   ```bash
   npm run build
   ```

## Testing

Run tests with:
```bash
npm test
```
