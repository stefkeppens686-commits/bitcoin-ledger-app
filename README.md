# Bitcoin Ledger App

A Bitcoin application for Ledger hardware wallets.

## Features

- Secure Bitcoin transaction signing
- Address management
- Balance tracking
- Multi-account support

## Installation

To build and install this application on your Ledger device, follow the instructions in the [INSTALLATION.md](INSTALLATION.md) file.
